API Documentation - Vendor Management System

Below are some guidelines for using the API endpoints:

  • Vendor Profile Management: Use the following endpoints to manage vendor profiles:
    • POST /api/vendors/
    • GET /api/vendors/
    • GET /api/vendors/{vendor_id}/
    • PUT /api/vendors/{vendor_id}/
    • DELETE /api/vendors/{vendor_id}/
  • Purchase Order Tracking: Use the following endpoints to track purchase orders:
    • POST /api/purchase_orders/
    • GET /api/purchase_orders/
    • GET /api/purchase_orders/{po_id}/
    • PUT /api/purchase_orders/{po_id}/
    • DELETE /api/purchase_orders/{po_id}/
  • Vendor Performance Evaluation: Use the following endpoint to retrieve a vendor's performance metrics:
    • GET /api/vendors/{vendor_id}/performance/
